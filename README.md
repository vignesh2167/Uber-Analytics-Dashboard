# Uber Analytics Dashboard 🚖

An interactive Uber Analytics Dashboard built using Python, Streamlit, Pandas, and Plotly.

## Features
- Ride booking analysis
- Revenue and trip insights
- Interactive charts and graphs
- User-friendly dashboard
- Real-time data filtering

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Dataset
Uber ride booking dataset (CSV/XLSX)

## Run Project
```bash
streamlit run app.py
